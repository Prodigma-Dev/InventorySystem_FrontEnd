Frequently Asked Questions

What is this admin template?

This admin template is a fully responsive and customizable dashboard built with Angular Material 19 and Material 3. It provides multiple themes, custom UI components, and flexible layouts for building modern web applications.

Can I customize the theme?

Yes! The template supports multiple themes and allows customization using Material 3 color palettes. You can modify styles in the SCSS files or adjust the theme configuration in the settings.

Does this template support authentication?

The template includes authentication UI components but does not have built-in backend authentication. You can integrate it with Firebase, Auth0, or any custom API for user management.

Is this template mobile-friendly?

Yes, it is fully responsive and optimized for desktop, tablet, and mobile devices.

Can I use this template for commercial projects?

Yes, as long as you follow the licensing terms. Please check the license file included with the template for details.

How do I get updates for this template?

You will receive updates and bug fixes through the platform where you purchased the template. Follow the changelog for new features and improvements.
